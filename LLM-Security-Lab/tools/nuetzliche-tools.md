# Nützliche Tools für LLM Security Testing

## Lokale LLM-Umgebung

### Ollama
- **Was:** Lokale LLM-Laufzeitumgebung
- **Wofür:** Modelle wie DeepSeek, Llama, Mistral lokal betreiben und testen
- **Link:** [ollama.com](https://ollama.com)
- **Status:** ✅ Installiert und konfiguriert

## Automatisierte Testtools

### NVIDIA Garak
- **Was:** Open-Source LLM Vulnerability Scanner
- **Wofür:** Automatisierte Schwachstellenanalyse von LLMs (Prompt Injection, Halluzinationen, Datenlecks)
- **Link:** [github.com/NVIDIA/garak](https://github.com/NVIDIA/garak)
- **Installation:** `pip install garak`
- **Status:** ⬜ Noch zu installieren

### Microsoft Counterfit
- **Was:** Adversarial ML Attack Framework
- **Wofür:** Automatisierte Angriffe auf ML/KI-Modelle testen
- **Link:** [github.com/Azure/counterfit](https://github.com/Azure/counterfit)
- **Status:** ⬜ Noch zu testen

## Lern- und Übungsplattformen

### Gandalf by Lakera
- **Was:** Kostenloses Browser-Spiel
- **Wofür:** Prompt Injection üben – Level für Level ein Passwort aus einer KI extrahieren
- **Link:** [gandalf.lakera.ai](https://gandalf.lakera.ai)
- **Status:** ⬜ Noch zu spielen

### Damn Vulnerable LLM Agent (DVLA)
- **Was:** Absichtlich unsichere LLM-Anwendung zum Üben
- **Wofür:** Praxisnahes Testen von LLM-Schwachstellen in einer kontrollierten Umgebung
- **Link:** Auf GitHub suchen nach "Damn Vulnerable LLM"
- **Status:** ⬜ Noch zu testen

## Dokumentation und Frameworks

### OWASP Top 10 for LLM Applications
- **Link:** [owasp.org/www-project-top-10-for-large-language-model-applications](https://owasp.org/www-project-top-10-for-large-language-model-applications/)

### Microsoft AI Red Team Practices
- **Was:** Offizielle Dokumentation von Microsoft über AI Red Teaming
- **Wofür:** Best Practices für das systematische Testen von KI-Systemen
- **Link:** In Microsoft Learn nach "AI Red Team" suchen

### MITRE ATLAS
- **Was:** Framework für Adversarial Threat Landscape for AI Systems
- **Wofür:** Systematische Kategorisierung von Angriffen auf KI-Systeme (wie MITRE ATT&CK aber für KI)
- **Link:** [atlas.mitre.org](https://atlas.mitre.org)

# Lernmaterial und Ressourcen

## Einstieg in LLM Security

### Grundlagen verstehen
1. **OWASP Top 10 for LLM Applications** lesen und verstehen → [Eigene Übersicht](../owasp-llm-top10/uebersicht.md)
2. **Gandalf by Lakera** durchspielen → Prompt Injection praxisnah lernen
3. **Eigenes Lab aufsetzen** → Ollama + DeepSeek (oder anderes Open-Source-Modell)

### Praktische Übungen (Reihenfolge)
1. ✅ System Prompt Extraction – direkte Methoden
2. ✅ Kreative Bypass-Techniken (Gedicht, Geschichte, Rollenspiel)
3. ✅ Cross-Lingual Prompt Injection
4. ⬜ Jailbreaking – Sicherheitsregeln des Modells umgehen
5. ⬜ Output Manipulation – schädlichen Code in Modell-Ausgaben einschleusen
6. ⬜ Indirekte Prompt Injection – über externe Dokumente angreifen
7. ⬜ NVIDIA Garak automatisierte Scans durchführen
8. ⬜ Eigene Schutzmaßnahmen entwickeln und testen
